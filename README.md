# Supply Chain Pulse

Executive supply chain performance dashboard for AbbVie, powered by Claude AI for real-time narrative insights and risk analysis. Built with Next.js 14, Tailwind CSS, Recharts, and the Anthropic API.

## Setup

```bash
npm install
cp .env.local.example .env.local
# Add your Anthropic API key to .env.local
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Connect to Vercel and add the `ANTHROPIC_API_KEY` environment variable.

## Note

All data is synthetic, modeled on pharma industry supply chain benchmarks.
